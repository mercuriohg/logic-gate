# Projeto Logic Gate

Este repositório contém um projeto de portas lógicas. O objetivo é demonstrar o comportamento de circuitos lógicos básicos, como AND, OR, NOT, NAND, NOR e XOR.

## Descrição

- Implementação de portas lógicas básicas.
- Possibilidade de testar combinações de entradas.
- Ideal para aprendizado de lógica digital e programação de circuitos.

## Como usar

1. Abra o projeto em seu editor de código ou IDE.
2. Verifique os arquivos de implementação e exemplos.
3. Execute o código conforme a linguagem e ambiente utilizados.

## Funcionalidades

- Simulação de portas AND e OR.
- Simulação de porta NOT.
- Implementação de portas compostas como NAND e XOR.
- Testes simples de combinações de entrada.
